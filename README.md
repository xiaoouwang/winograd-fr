## Join us

### The Git/correct way

* Fork the project.

* Create a file under `source/log` with your name, using `emma_template.rst` of Emma Dubois as a template.

* Edit the your rst file to log your work. It's quite straightforward.

* Make a commit and pull request.

**emma_template.rst**

```md

====================
```
